Card

Display a card with content with text content or copy.

Preview

Card 1

This card has a small amount of content.

Card 2

This one has a bit more content than the first card. It demonstrates how the height balances out across all the cards.

Card 3

This is the longest card. It has the most content of the three, ensuring that the height is determined by this one and the others stretch to match.

Extra paragraph just to make it taller!

Code

                
<div class="cards">
    <div class="card">
        <h3>Card 1</h3>
        <p>This card has a small amount of content.</p>
    </div>
    <div class="card">
        <h3>Card 2</h3>
        <p>This one has a bit more content than the first card. It demonstrates how the height balances out across all the cards.</p>
    </div>
    <div class="card">
        <h3>Card 3</h3>
        <p>This is the longest card. It has the most content of the three, ensuring that the height is determined by this one and the others stretch to match.</p>
        <p>Extra paragraph just to make it taller!</p>
    </div>
</div>        
                
            
                
.cards {
    display: flex;
    flex-direction: column;
    gap: 1rem;
}

/* Tablets and above */
@media (min-width: 900px) {
    .cards {
        flex-direction: row;
    }
}

.card {
    flex: 1; /* Equal height and width */
    display: flex;
    flex-direction: column;
    padding: 1rem;
    border-width: 1px;
    border-style: solid;
    border-color: #333;
    border-radius: 8px;
    background: #171717;
    color: #fafafa;
}
